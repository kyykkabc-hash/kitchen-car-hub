# ドコデル スキル集

## スキル：イベント検索してDB登録

### 呼び出し方
「イベントを検索して」と言われたら、このスキルを実行する。

### 手順

1. 今日の日付から3ヶ月後までの土日・祝日をリストアップする

2. 各日付について以下のキーワードで検索する：
   - "YYYY年M月D日" "マルシェ" "出店募集"
   - "YYYY年M月D日" "イベント" "出店募集"
   - "令和X年M月D日" "マルシェ" "出店募集"

3. 検索結果から以下の情報を抽出して一覧表示する：
   - イベント名
   - 開催日
   - 都道府県
   - 市区町村
   - 会場名（公園名・施設名など正式名称）
   - 会場の住所（できるだけ詳細に）
   - イベント概要URL
   - 応募フォームURL（あれば）
   - キッチンカー募集か・テント出店募集か・両方か

4. 一覧を裕一さんに見せて「どれを登録しますか？」と確認する

5. 指定されたイベントだけSupabaseのeventsテーブルに登録する：
   - title: イベント名
   - event_date: 開催日
   - prefecture: 都道府県
   - location_name: 会場名
   - address: 会場の住所
   - event_url: イベント概要URL
   - application_url: 応募フォームURL
   - status: 'published'
   - vendor_type: 'kitchen_car' / 'tent' / 'both'

### 注意事項
- 情報が不足している場合は「不明」と入力する
- 重複登録しないよう、既存データと照合する
- 一度に登録しすぎず、内容を確認してから進む
- 他サイトの著作権に配慮し、URLと基本情報のみ収集する
